This is the Dreams 2020 embed testing page.

We're going to embed the chart below.

The embed code is as follows:

  1. In the <head> of your page, include our CSS and JavaScript bundles:

    <script src="https://dreams-2020.netlify.app//static/js/main.js"></script>
    <link rel="stylesheet" href="https://dreams-2020.netlify.app//static/css/main.css" /> Make sure to give the embed div sensible height and width styles, based on your page. E.g.: 

                    #embed {
                    width: 800px;
                    height: 600px;
                    max-width: 100%;
                    margin: auto;
                }

  2. Include an empty <div> on your page, with an ID so you may select it from code. e.g.: <div id="embed" ></div>

  3. Run our embed script in a <script> tag, e.g.

    <script> window.embedDreamChart("#embed"); </script>

    The above function, window.embedDreamChart, accepts 2 parameters:

    • A selector string (e.g. "#embed"). This refers to the empty div you made.
    • Optionally, an options parameter to elect which graph to show, and whether to allow toggling to other graphs. e.g: {activeChart: "area", showAll: true} If you don't add this parameter, it will default to the bar chart and toggleable.
